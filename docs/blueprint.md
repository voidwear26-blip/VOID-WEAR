# **App Name**: VOID WEAR

## Core Features:

- Immersive Hero Experience: Fullscreen hero with a slowly rotating, glowing eclipse logo, subtle particle effects, and cinematic text animations on entrance.
- Interactive Product Display: Grid and collection layouts featuring products that animate on scroll, with glowing, scaling, and tilting effects on hover for 3D depth.
- Product Detail Page: Dedicated pages for individual products, offering detailed information, image zoom, and 'Add to Cart' functionality.
- Shopping Cart Management: A sliding cart drawer allowing users to add/remove items, adjust quantities, and view their selections with smooth transitions.
- AI-Powered Style Assistant: A generative AI tool to assist users by creating unique outfit combinations or style recommendations based on 'VOID WEAR' products.
- Seamless Site Navigation: Implement smooth scrolling (e.g., via Lenis) and sophisticated page transitions (e.g., via Framer Motion) for a fluid user journey.
- Product & User Data Storage: Essential database functionality to store product details (names, descriptions, images, pricing) and basic user account information and order history.

## Style Guidelines:

- Scheme: Dark. Primary (glowing elements): dark grey(). Background: Pure black (#000000). Accent (interactive elements/highlights): Vibrant white ().
- Headline and Body text: 'Space Grotesk' (sans-serif) for a clean, futuristic, and geometric aesthetic. All text should be uppercase with wide letter spacing.
- Utilize simple, geometric social icons that emit a soft glow effect on hover, aligning with the ethereal, high-end theme.
- Minimalist, spacious design with generous padding to create a 'breathable' feel. Elements should stand out through luminous effects rather than borders, set against the pure black background.
- Implement a 'cinematic entrance' for the hero, parallax scrolling, elements revealing with fade + slide animations, glowing hover states, button inversions, magnetic cursor interactions, and subtle animated background effects (particles/nebula gradient).
